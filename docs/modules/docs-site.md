# docs-site

User-facing prose documentation (rendered via Jekyll on GitHub Pages),
the project README, the auto-generated file-level index, and the shared
`CLAUDE.md` efficiency guidance loaded by all subagents.

## Entry points
- `README.md` — Project overview and usage guide.
- `CLAUDE.md` — Shared efficiency guidance for all subagents.
- `CODEBASE_INDEX.md` — Auto-generated file-level index.
- `LICENSE` — MIT license text.
- `docs/index.md` — Documentation site landing page.
- `docs/architecture.md` — Pipeline and system architecture.
- `docs/agents.md` — Agent definitions and pipeline-phase mapping.
- `docs/cli.md` — CLI reference for `cai.py` subcommands.
- `docs/configuration.md` — Environment variables and configuration.
- `docs/fsm.md` — Auto-generated lifecycle FSM diagrams.
- `docs/_config.yml` — Jekyll configuration.
- `docs/modules.yaml` — Module index (owned by cai-review-docs).
- `docs/modules/*.md` — Per-module narratives (including this file).
