# agents-utility

Utility subagents — memory curator, git command runner, cost optimiser,
and weekly external-library scout.

## Entry points
- `.claude/agents/utility/cai-memorize.md` — Post-solved memory curator.
- `.claude/agents/utility/cai-git.md` — Git command runner used by other agents.
- `.claude/agents/utility/cai-cost-optimize.md` — Weekly cost-reduction proposer.
- `.claude/agents/utility/cai-external-scout.md` — Weekly OSS library scout.
