# scripts

Maintenance and code-generation scripts invoked manually or from CI.

## Entry points
- `scripts/generate-index.sh` — Generator for `CODEBASE_INDEX.md`.
- `scripts/generate-fsm-docs.py` — Generator for `docs/fsm.md` (Mermaid).
- `scripts/server-cleanup.sh` — Server-side age/size cleanup for the transcript-sync store.
