# agents-review

Subagent definitions that review open PRs before merge — ripple-effect
review, documentation review, merge-readiness assessment, and PR-comment
filtering.

## Entry points
- `.claude/agents/review/cai-review-pr.md` — Ripple-effect inconsistency review.
- `.claude/agents/review/cai-review-docs.md` — Docs-drift review.
- `.claude/agents/review/cai-merge.md` — Merge-readiness verdict.
- `.claude/agents/review/cai-comment-filter.md` — Resolved / unresolved PR-comment classifier.
